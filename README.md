# 1KZARIKI · «Зарики»

Счётчик для игры в **1000 на кубиках** (пять костей). Заводишь партию, добавляешь игроков, щёлкаешь очки за каждый кон — а бочка, ямы, самосвалы, заходы и штрафы считаются и подсвечиваются сами.

**▶ Открыть приложение: https://vibebrainsprojects.github.io/1KZariki/**

Один файл `index.html`, без сборки, всё хранится в браузере. Открыл — играешь.

---

## Возможности

- **Партии и дашборды** — список партий, у каждой таблица «коны × игроки» с итогами.
- **Настройка правил кнопками** — бочка, шкала очков, ровно 1000, ямы, штраф за самосвал, 3 болта подряд, обгон, порог входа и режим входа. Плюс пресеты в один тап.
- **Ввод с телефона** — тап по клетке открывает модалку с цифровой клавиатурой, быстрые комбинации и кнопка «самосвал». Имя игрока — модалка с раскладкой ЙЦУКЕН / QWERTY / 123.
- **Подсветка состояний** — корона лидеру, плашка «на бочке» с авто-расчётом «−N до 1000», маркер ямы, красный самосвал, строка штрафов, статус «зайти от 120».
- **Правка и удаление** — любой счёт правится тапом по клетке; коны удаляются в режиме правки; партии — из списка. Везде свой диалог подтверждения.
- **Статистика** — победы, выходы на бочку, самосвалы, рекордный кон по игрокам (кросс-партийно).
- **Бэкап и облако** — экспорт/импорт JSON и синхронизация через приватный GitHub Gist по токену. Чистый фронт, без бэкенда.
- **Адаптивность** — мобайл-фёрст, ориентация значения не имеет.
- **PWA** — устанавливается на домашний экран, запускается в standalone и работает офлайн (манифест + service worker, кэш app-shell и CDN).

## Быстрый старт

```bash
# вариант 1 — просто открыть
open index.html            # или двойной клик

# вариант 2 — отдать статикой (любой статический сервер)
python3 -m http.server 8080
# → http://localhost:8080
```

Для деплоя достаточно положить `index.html` (со спутниками) на любой статический хостинг (GitHub Pages, Netlify, S3 и т.п.).

### Деплой на GitHub Pages

Сайт — статика в корне репозитория, поэтому деплоится прямо из ветки, без CI: **Settings → Pages → Source: «Deploy from a branch» → Branch `main`, Folder `/ (root)` → Save**. Файл `.nojekyll` в корне отключает обработку Jekyll, чтобы ассеты (`sw.js`, `icons/`) отдавались как есть. Сайт публикуется на `https://vibebrainsprojects.github.io/1KZariki/`.

## Стек

- **React 18** + **ReactDOM** + **Babel Standalone** (JSX компилируется в браузере) — подключаются с cdnjs.
- Ванильный **CSS** с дизайн-токенами, без UI-фреймворка.
- Шрифты Google Fonts: Unbounded (бренд), Manrope (текст), JetBrains Mono (цифры), Caveat (мел).
- Хранилище — `localStorage` с откатом в память (работает даже в песочнице без доступа к стораджу).

Сборки нет. Это сознательное решение для v1 — см. план миграции на Vite в [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md).

## Правила в двух словах

5 костей, цель — первым набрать ровно **1000**. Сами по себе зачётные грани только `1` и `5`, тройки считаются по номиналу. На отметке 880 (или 850) — **бочка**: счёт замораживается, нужно закрыться ровно за один ход. Полный разбор и сопоставление с настройками — в [`docs/RULES.md`](docs/RULES.md).
